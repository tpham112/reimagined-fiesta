    Adding Inactive Programs for Students¶

    Only for students with catalog rights to older years (i.e., continuing students since a specific year).

    SHAP: 1. Add application for the older semester and older catalog. 2. Add comments in ASUM regarding why the application was added to push the program.

    Add Team Members to Reg Group for RGN Permissions¶

    RGUS: 1. Select Reg group 1reg_lead. 2. Add employee ID to the list. 3. Save All and have them test.

    Athletic Interest Survey¶

    Email: staffsupportccctc@openccc.zendesk.com

    Backdating Drops for Student Business Office¶

    STAC: 1. Take screenshot of the course and save for reference. 2. Backdate the drop to the first day of class and remove the grade if necessary. 3. Save All. 4. Let SBO know to rebill. 5. Once rebill is complete, fix the drop date back to the original date and re-enter the grade if necessary.

    Batch Enrollment¶

    SECT: Check build date — do not batch on the same day as it is built.

    SLED: 1. Enter a Saved List name → OK. 2. Click Add (for new list). 3. Detail into the first line. 4. Paste in ID numbers → Save. 5. Verify amount of IDs is correct. 6. Click Save All.

    SCBL: Use previously created list. Enter class name and section number. (Multiple sections can be enrolled simultaneously if same roster AND same section dates.)

    BRGN:

    Field Value
    Section Blocks Name used in SCBL
    RG User Group ARSTAFF
    Update No first to test
    Saved List Name Name used in SLED

    Critical errors requiring removal from SLED: Require Overload Petition / Repeat / Does not meet prereq / Time conflict.

    Run again with Update: Yes to enroll.
