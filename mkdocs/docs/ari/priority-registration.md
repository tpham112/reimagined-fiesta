# Student Parent/Guardian — Priority Registration

## Overview

Processing Dynamic Form submissions for student parent/guardian priority registration.

## Processing Steps

### 1. Access the Student Record
- Log in to **Colleague**.
- Go to **COAF** for the student listed on the Dynamic Form submission.

---

### 2. Add Priority Registration Code
- Navigate to a **new line** in COAF.
- Enter **`1SPPR`** and press **Enter**.

---

### 3. Enter Membership Period
- In the **Membership Periods** fields:
    - Enter the **Start Date** for the current term.
    - Enter the **End Date** for the current term.

---

### 4. Save Changes
- Click **Save All** to apply the update to the student's record.

---

## Completion

- Confirm the Priority Registration code displays correctly for the specified term.
- Update the Dynamic Form status as appropriate.
